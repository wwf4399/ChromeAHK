# ChromeAHK
谷歌浏览器-AutoHotKey插件

测试环境：win11 + Chrome(149.0.7827.115（正式版本） （64 位）)

如何使用:
1.直接使用我编译好的.exe
2.下载autoHotKey V2，然后打开.ahk后缀的文件

功能：
1.右键网页上面的选项卡[标题]可以关闭标签页 


https://github.com/user-attachments/assets/0816f4e4-2e51-407f-8451-786ee365e073


2.右键浏览器窗口右上角的浏览器关闭按钮，可关闭当前标签页 


https://github.com/user-attachments/assets/0a56f4e5-5695-43ea-b7f9-2a4cefd5ef82



3.右键选项卡列表后面的“新建标签页”按钮可直接搜索复制内容、或打开复制的网址


https://github.com/user-attachments/assets/5192636c-4584-4276-a00e-af02b8164039






4.左键单击书签可在新窗口打开


https://github.com/user-attachments/assets/ae50e642-8b6b-4bca-b100-3487a33d28fc
